Portfolio Projects

Check out a few of the projects I have completed as a part of the MERN Full Stack Development Course

PacMen Exercise



This project manipulates PacMan images to simulate the PacMan moving across the screen and reversing direction. This project is intended to assist with learning how to control the Document Object Model.

Eye Exercise


This project utilizes a combination of HTML, CSS and JavaScript by displaying a pair of eyes (styled with CSS) on a single webpage (structred with HTML) that are programmed to follow the movement of the mouse on the screen (programmed with JavaScript).

Bus Tracker Exercise


This project sources real-time data of Boston MBTA busses to display the live locations of the busses on a map by showing the stops between MIT and Harvard illustrated by a pin.
